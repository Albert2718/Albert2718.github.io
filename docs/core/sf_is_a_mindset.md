最近天天晚上回宿舍就打SF6，慢慢发现，这款游戏带给我的不只是乐趣。

街头霸王6的游戏性当然没得说，它是一种理性和感性结合起来的艺术。感性上，杰米的超级爆爆回再搭配上鸳鸯戏的BGM被多少人称为街霸6的招生办，更不用说里面的各种各样的角色以及震撼的SA3的画面表现力。理性上，这个游戏的规则、深度和策略等等更是无与伦比。从最基础的以帧为单位的逻辑再到阅读对手后的心理博弈，这是街霸6所代表的格斗游戏一直以来所拥有的理性的魅力。

我非常感激这款游戏，它治愈了我的浮躁，赠与我了一种很理想的心态。

SF6的正反馈多种多样，从最开始训练营搓出一个小招或者一个大招的欣喜，再到能熟练地打出一套连招，以及慢慢的能在实战中运用出自己训练的成果，这些是无关他人，客观的满足感与正反馈。

更重要的是，在对弈中，慢慢明白所谓胜利与失败，都是为我所用的，是由我来定义的。

固然每场比赛都有它客观上的胜负，但是每次对局的过程，才是我该定义胜利的地方。在开始时，今天的目标就是打出一套新学的连段，或者在对手的某个招式上找到破绽并反制它。只要我达成了这个目标，我都会很开心。
